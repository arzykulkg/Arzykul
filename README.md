# Арзыкуловтордун Тукум Дарагы

## Arzykul Family Tree - Kyrgyz Cultural Interactive Genealogy

An interactive, beautifully designed family tree application showcasing the Arzykul family genealogy with Kyrgyz cultural aesthetics.

### 🌿 Features

- **Interactive Tree Navigation**: Click nodes to expand/collapse family branches
- **Automatic Pan & Zoom**: Smoothly centers on selected family members
- **Search Functionality**: Find family members by name (Kyrgyz language support)
- **Responsive Design**: Works on all screen sizes
- **Cultural Design**: Traditional Kyrgyz colors and patterns
  - Golden accents (#D4A574)
  - Warm earth tones
  - Geometric patterns

### 🎨 Design Elements

- **Color Palette**:
  - Primary: Gold/Altın - #D4A574
  - Accent: Turkuaz - #1F7BA0
  - Female Members: Deep Red - #C41E3A
  - Background: Warm Brown - #1a1410

- **Typography**: 
  - Playfair Display (Serif) for names
  - Raleway for UI text

### 🚀 Usage

1. Open `index.html` in your web browser
2. Click on any family member to expand their descendants
3. Use search box to find specific people
4. Zoom controls on the right: +/− buttons
5. Fit All button (⊡) to view entire tree
6. Reset button (⟳ Баштапкы) to return to starting view

### 📱 Features

- **Pan & Drag**: Click and drag to move around the tree
- **Scroll Zoom**: Use mouse wheel to zoom in/out
- **Touch Support**: Mobile-friendly with touch gestures
- **Auto-Center**: When expanding nodes, view automatically centers on selection

### 📊 Data Structure

The family tree contains:
- **246** total family members
- **5** main branches
- Multi-generational tracking
- Gender indicators (F = Female)

### 🔧 Technical Stack

- Pure HTML5 with SVG for connections
- Vanilla JavaScript (no dependencies)
- CSS3 with custom properties
- Responsive and adaptive layout
